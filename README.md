# Objetivo do Sistema
Sistema ERP de Estoque, voltado para o controle de estoque e produtos, juntamente com suas respectivas funcionalidades e implementações.

## Funcionalidades

### Produtos
- Cadastrar produto
- Editar produto
- Inativar produto
- Buscar produto

### Estoque
- Entrada de estoque
- Saída de estoque
- Consultar saldo

## Movimentações
Todas as movimentações devem ficar registradas.

## Arquitetura
Esse sistema será implementado em Layered Architecture
